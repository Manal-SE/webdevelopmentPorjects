CCSW-321 Syllabus

Title of module		CCSW-321 Web Development

Credit value  		3

Prerequisite 		CCCS 214

Designation		Required

Course Description    
This course provides an overview
of Internet application development.
 
Aims		    
  • Study Internet programming and Web application development.
  • Learn basic principles for building Internet applications.
  • Introducing client-side and server-side programming.
 
Level	              
6
 
Class Schedule	      
Meet 50 minutes 3 times/week (3 lectures)
